# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the technical merits. Harassment, discrimination, and personal attacks are not tolerated.

Report unacceptable behavior to **conduct@riga.solutions** (RIGA Solutions, the project maintainer, hosts conduct intake on behalf of the project). Reports are handled confidentially. Maintainers who do not follow or enforce the Code of Conduct may be removed from the project.

The full text of the Contributor Covenant 2.1 governs; this file is a pointer to it.
